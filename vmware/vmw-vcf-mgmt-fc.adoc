= Deployment workflow for converting vCenter server instances to VCF management domains with Fibre Channel datastores
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ../media/

[.lead]
Convert an existing vSphere 8 cluster with NetApp ONTAP Fibre Channel datastores to a VMware Cloud Foundation management domain. This workflow includes reviewing configuration requirements, deploying ONTAP tools and provisioning FC datastores, and using the VCF Import Tool to validate and convert the cluster.

For an overview of the conversion process, see the VMware documentation: https://techdocs.broadcom.com/us/en/vmware-cis/vcf/vcf-5-2-and-earlier/5-2/map-for-administering-vcf-5-2/importing-existing-vsphere-environments-admin/convert-or-import-a-vsphere-environment-into-vmware-cloud-foundation-admin.html[Convert a vSphere environment to a management domain or Import a vSphere environment as a VI workload domain in VMware Cloud Foundation].
